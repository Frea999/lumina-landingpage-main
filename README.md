
# Lumina landingpage eksamen

Det her e mit eksamens projekt for lumina landingpage

# HTML
I min html har jeg fjernet div tags og skiftet dem ud med semantiske tags. jeg har også givet nogle klasser for at kunne lave hover effekter og tilføjet en ny secion til se hele vores udvalg

# CSS

I mit css har jeg lavet hovereffect, og stylet den section hvor man kan se hele udvalget. Der er også lavet meget få smårettelsere

# JS 
I mit javascript har jeg lavet en funktion der gør at når man klikker på skift farve knappen, så komemr der en højtaler frem i en ny farve.
